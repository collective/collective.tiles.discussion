===========================
collective.tiles.discussion
===========================

Tile for showing last discussion items in the portal

Features
--------

- Can be bullet points


Installation
------------

Install collective.tiles.discussion by adding it to your buildout::

    [buildout]

    ...

    eggs =
        collective.tiles.discussion


and then running ``bin/buildout``.
